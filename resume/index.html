曬裝備

方便分享的高山裝備管理功能網站。
在 Pace Design 的同事 Ruoshin 在初期就加入，並邀請後端工程師 Ruby 一起揮灑創作！

  • 接受 Ruoshin 建議我學習寫 end to end 測試。
  • 設計了我個人最活躍的 Design system。
  • 加強非同步溝通。

Kuaitshiu

2019 — 2021

土木營建專案使用的資訊功能網站。
需要考量包含業主、承包商不同角度的使用者需求以及法律合規。

  • 讓內部理解為什麼以及如何訪談使用者。
  • 升級開發流程：從原生 PHP 改用 Laravel、採用 GIT 跟自動部署。
  • 執行 Style tile 讓內部輕鬆並體會設計介面的考量。

App Apeh

2019 — 2021

土木業與軟體設計跨領域小團隊的品牌設計。
這個團隊只有三個人，是為了要開發土木營建功能網站而成立的。
設計上的挑戰是如何讓團隊內部達到對外呈現的共識。

Saviah

2017 — 2018

翻修專為公益團體使用的捐款管理功能網站。
原本的版本被使用很多年，但我觀察到客服人員還是經常以很長的電話輔助使用者，所以有客服同事可以訪問對我來說很有幫助。
這個專案在開發部分也頗具挑戰，例如仍使用手動 FTP 部署以及未採用版本控制。

  • 拆解原本功能對於使用者的意義是什麼。
  • 與 MIS 同事合作架了 GitLab 在公司內部推廣。

Nucleus

2016 — 2017

這是一個收集資料的功能網站，讓內部分析師可以收集大量的社交平台上的資訊以及新聞，並加上自訂的關鍵字組合，這種組合要可以很複雜。
分析師得以從收集到的資料再製作成報告發送 email 給客戶。
但是我得誠實的說，我當時的能力遠低於這個專案所需要的，所以身心靈鼻青臉腫。
不過我至少在這裡學會 GIT 基本操作。
